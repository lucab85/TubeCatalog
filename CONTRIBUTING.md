1. **Fork the repo**
2. **Clone your fork**
   ```bash
   git clone https://github.com/lucab85/TubeCatalog.git
   cd TubeCatalog
````

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Create a `.env` file**

   ```env
   YOUTUBE_API_KEY=your_youtube_api_key
   OPENAI_API_KEY=your_openai_api_key
   ```

5. **Start the dev server**

   ```bash
   npm run dev
   ```
